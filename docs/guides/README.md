## Quick Start

### Prerequisites

- Node.js 18+
- Rust 1.70+
- Solana CLI 1.17+
- Anchor 0.29+

### Installation

```bash
# Clone repository
git clone https://github.com/NakaSato/gridtokenx-anchor.git
cd gridtokenx-anchor

# Install dependencies
pnpm install

# Build programs
anchor build

# Run tests
anchor test
```
